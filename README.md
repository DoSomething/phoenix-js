Javascript client for the DoSomething.org Phoenix API.

## Testing

To run `npm test` locally, create a `.env` and set config variables:
````
PHOENIX_REST_API_BASEURI
PHOENIX_REST_API_USERNAME
PHOENIX_REST_API_PASSWORD
````
